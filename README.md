# ZX-Spectrum-Key-Labels
ZX Spectrum labels for your keyboard.

These are designed to be printed on half inch letter-sized label stock, in particular, [OL2050WR from OnlineLabels](https://www.onlinelabels.com/products/ol2050wr).

There are two versions of the PDF, one with regular text, the other with the text replaced with outlines. If needed, you should be able to edit either PDF in Adobe Illustrator or Inkscape.
